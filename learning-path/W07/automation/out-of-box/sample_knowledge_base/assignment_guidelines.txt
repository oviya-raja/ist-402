W07 Assignment Guidelines

ASSIGNMENT OVERVIEW
===================

Objective: Automate workflows using OpenAI Agents to improve efficiency and reduce manual effort.

Instructions: Develop and deploy an AI agent using OpenAI Agent Builder tools that integrates with selected APIs and services.

DELIVERABLES (100 Points Total)
================================

1. Functional OpenAI Agent (20 pts)
   - Agent is fully functional
   - Demonstrates intelligent decision-making
   - Proper prompt engineering
   - Appropriate tools and integrations
   - Handles edge cases gracefully
   - Clear documentation of capabilities
   - Evidence of thorough testing

2. Documented Workflows (20 pts)
   - Comprehensive workflow documentation
   - Clear step-by-step automation processes
   - Detailed diagrams (triggers, actions, data flow)
   - Integration points documented
   - Error handling strategies
   - Workflow optimization notes

3. GitHub Repository (20 pts) - Optional
   - Well-organized structure
   - Meaningful commits
   - Comprehensive README
   - Team member details
   - Professional formatting

4. Screenshots (20 pts)
   - Agent Builder configuration
   - Tools/functions setup
   - Prompt instructions
   - Memory settings
   - Testing/execution
   - Evidence of deployment

5. Final PDF Report (20 pts)
   - Project overview and objectives
   - Workflow identification and justification
   - Implementation details
   - Team roles and responsibilities
   - Challenges faced and solutions
   - Results and testing outcomes
   - Future improvements
   - All screenshots embedded and labeled

EXAMPLE WORKFLOWS
================

1. Automating student query responses using a knowledge base
2. Scheduling meetings using calendar API
3. Summarizing emails or documents

SUBMISSION CHECKLIST
====================

Before submitting:
- [ ] Functional agent deployed and tested
- [ ] Workflow documentation complete with diagrams
- [ ] GitHub repository organized (if applicable)
- [ ] Screenshots of Agent Builder setup captured
- [ ] Final PDF report prepared and reviewed
- [ ] All team members have contributed
- [ ] Code is properly commented and documented
- [ ] All deliverables meet rubric requirements
