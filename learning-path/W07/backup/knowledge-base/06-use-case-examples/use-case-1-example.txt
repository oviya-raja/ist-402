Job Fitment Analysis Agent - Knowledge Base

EXAMPLE SCENARIO 1
==================

User Input:
-----------
"I'm a Computer Science student graduating in May 2025. I have experience with Python, Java, AWS, and SQL. I'm looking for software engineering roles at Cisco and Google. I prefer remote work and entry-level positions."

Agent Processing:
-----------------
1. Parse company priorities: Priority 1 (Cisco, Google)
2. Extract student profile:
   - Skills: Python, Java, AWS, SQL
   - Education: BS CS (expected May 2025)
   - Experience: Student (internships/projects)
   - Location: Remote
   - Level: Entry-level

3. Retrieve from knowledge base:
   - Cisco typical entry-level requirements
   - Google typical entry-level requirements
   - Common software engineering roles

4. Analyze fitment for typical roles:
   - Software Engineer - New Grad (Cisco)
   - Software Engineer I (Google)
   - Associate Software Engineer (Google)

Agent Output:
-------------
"Great! I can help you find roles that match your profile. Based on your skills (Python, Java, AWS, SQL) and your target companies, here are roles that would be a good fit:

PRIORITY 1 COMPANIES:

Cisco - Software Engineer - New Grad
Fitment: 85% ✅ Strong Match
Why it matches:
- ✅ Python and Java match requirements
- ✅ AWS experience aligns with cloud focus
- ✅ SQL skills match database needs
- ✅ Entry-level/new grad program
- ⚠️ May need networking basics (can learn)
Job Search: https://careers.cisco.com/global/en
Search terms: "Software Engineer" + "New Grad" + "Entry Level"

Google - Software Engineer I
Fitment: 82% ✅ Good Match
Why it matches:
- ✅ Python and Java are commonly used
- ✅ Strong fundamentals needed (you have)
- ⚠️ May need stronger algorithms practice (for interviews)
- ⚠️ System design knowledge helpful
Job Search: https://careers.google.com/jobs/
Search terms: "Software Engineer" + "Entry Level" + "New Grad"

Google - Associate Software Engineer
Fitment: 78% ✅ Good Match
Why it matches:
- ✅ Designed for entry-level
- ✅ Python/Java experience relevant
- ⚠️ Interview process is competitive (prepare well)

RECOMMENDATIONS:
1. Focus on these roles first (highest fitment)
2. Prepare for technical interviews (algorithms, system design)
3. Highlight your AWS and SQL experience
4. Consider learning networking basics for Cisco roles

Would you like me to:
