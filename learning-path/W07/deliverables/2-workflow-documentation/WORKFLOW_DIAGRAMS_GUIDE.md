## üé® How to Render Mermaid Diagrams

### Method 1: Mermaid Live Editor (Easiest)

1. **Go to:** https://mermaid.live
2. **Open** any `.mmd` file from `workflow-diagrams/` folder
3. **Copy** the entire content
4. **Paste** into Mermaid Live Editor
5. **Diagram renders automatically**
6. **Export:**
   - Click "Download PNG" for images
   - Click "Download SVG" for vector graphics
   - Or copy as image

**Time:** 2-3 minutes per diagram

---

### Method 2: VS Code Extension

1. **Install Extension:** "Markdown Preview Mermaid Support"
2. **Open** `.mmd` file in VS Code
3. **Preview** using Markdown preview
4. **Export** using screenshot or extension tools

---

### Method 3: GitHub/GitLab (Automatic)

1. **Create** markdown file (e.g., `diagrams.md`)
2. **Add** mermaid code block:
   ```markdown
   ```mermaid
   [paste mermaid code here]
   ```
   ```
3. **View** on GitHub/GitLab - renders automatically
4. **Screenshot** or use browser tools to save

---

### Method 4: Draw.io / Lucidchart (Recreate)

1. **Use** Mermaid diagrams as reference
2. **Recreate** in Draw.io or Lucidchart
3. **Export** as PNG or PDF
4. **More control** over styling and layout
