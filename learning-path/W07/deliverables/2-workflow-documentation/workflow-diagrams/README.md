## How to Use These Diagrams

### Option 1: Mermaid Live Editor (Recommended)
1. Go to: https://mermaid.live
2. Copy content from `.mmd` file
3. Paste into editor
4. Export as PNG or SVG
5. Save to this folder or use in documentation

### Option 2: VS Code Extension
1. Install "Markdown Preview Mermaid Support" extension
2. Open `.mmd` file
3. Preview and export

### Option 3: GitHub/GitLab
- Mermaid diagrams render automatically in markdown
- Can be included directly in documentation

### Option 4: Draw.io / Lucidchart
- Use diagrams as reference
- Recreate in your preferred tool
- Export as PNG/PDF
