# Assignment Deliverables

This folder contains all files required for assignment submission.

## Folder Structure

- **1-functional-agent/** - Agent capabilities, limitations, use cases, testing evidence
- **2-workflow-documentation/** - Workflow diagrams, integration points, technical specs
- **3-github-repository/** - GitHub repo link or local copy (if using)
- **4-screenshots/** - All Agent Builder setup screenshots organized by category
- **5-final-report/** - Final PDF report and source files

## Submission Checklist

- [ ] Deliverable 1: Functional Agent documentation complete
- [ ] Deliverable 2: Workflow documentation and diagrams complete
- [ ] Deliverable 3: GitHub repository ready (if using)
- [ ] Deliverable 4: All screenshots captured and organized
- [ ] Deliverable 5: Final PDF report complete

---

*Organize your submission files here. The `ai-driven/` folder contains working files only and should NOT be submitted.*
