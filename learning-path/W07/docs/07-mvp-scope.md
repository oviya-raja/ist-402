# MVP Scope

## Phase 1: MVP (Minimum Viable Product)

### Scope
- **Target Companies:** Focus on 2-3 companies (e.g., Cisco, Google)
- **Primary Workflow:** Implement Workflow 1 (Job Search & Fitment Analysis)
- **Fitment Calculation:** Basic skills match percentage
- **Knowledge Base:** Simple student profile storage
- **Use Cases:** Use Case 1 (Find Matching Jobs) and Use Case 2 (Check Qualification)

### Deliverables
- ✅ Functional agent deployed in OpenAI Agent Builder
- ✅ Basic job search and fitment analysis
- ✅ Simple knowledge base integration
- ✅ Basic error handling

### Success Criteria
- Agent successfully searches 2-3 company job sites
- Returns accurate fitment percentages (±5% tolerance)
- Handles common errors gracefully
- Response time < 2 minutes per job
