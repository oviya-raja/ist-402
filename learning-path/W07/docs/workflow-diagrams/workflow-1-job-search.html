OpenAI Agent Builder Setup Instructions

Step 1: Create New Workflow

  1. Go to OpenAI Agent Builder
  2. Click "Create Workflow" or "New Workflow"
  3. Name it: "Job Search & Fitment Analysis"

Step 2: Add Trigger Node

  1. Drag "Agent" node from Core section to canvas
  2. Configure as trigger: "User Input"
  3. Set input schema: profile (skills, experience, education, target_companies)

Step 3: Add Validation Logic

  1. Drag "If / Else" node from Logic section
  2. Connect from User Input node
  3. Set condition: Check if profile has required fields
  4. True branch: Continue to Load Companies
  5. False branch: Connect to End node with error message

Step 4: Add Data Loading

  1. Drag "Transform" node from Data section
  2. Configure to read target-companies.csv
  3. Filter by status='active'
  4. Match user's target companies

Step 5: Add Web Scraping Tool

  1. Drag "File search" or custom tool from Tools section
  2. Configure web scraping function for job sites
  3. Set up for each company in the list

Step 6: Add Analysis Agent

  1. Drag "Agent" node from Core section
  2. Name: "Profile Match Analysis Agent"
  3. Configure system prompt for job analysis
  4. Connect knowledge base for profile matching
System Prompt: You are a Job Fitment Analysis Agent. Analyze job postings against student profiles. Calculate fitment percentages based on skills, experience, and education matches.

Step 7: Add Calculation Logic

  1. Add "Transform" nodes for fitment calculation
  2. Calculate: (matched_skills / total_required_skills) × 100
  3. Add skill gap identification logic
  4. Add ranking logic (sort by fitment, apply priority weighting)

Step 8: Add End Node

  1. Drag "End" node from Core section
  2. Configure output format: JSON with ranked jobs, fitment scores, skill gaps
  3. Connect from Rank Results node

Step 9: Test Workflow

  1. Click "Test" button in Agent Builder
  2. Provide sample profile input
  3. Verify each step executes correctly
  4. Check output format and accuracy
