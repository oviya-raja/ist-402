# Project Overview

**Project Name:** Job Fitment Analysis Agent  
**Technology:** OpenAI Agent Builder  
**Objective:** Automate job search and fitment analysis workflows for students

---

## Project Description

This project develops an AI agent using OpenAI Agent Builder to automate the job search and fitment analysis process for final year students. The agent will search company job sites, analyze job postings against student profiles, and provide fitment percentages with skill gap identification.

## Project Context

**Course:** IST-402 - AI Agents Development  
**Assignment:** W7 - Group Assignment: Agents Development Using OpenAI  
**Team:** Group project  
**Project Type:** Group assignment for course IST-402

## Technology Stack

- **Agent Framework:** OpenAI Agent Builder
- **Web Scraping:** Playwright, BeautifulSoup
- **Data Storage:** CSV files for configuration, knowledge base for profiles
- **Analysis:** LLM-based fitment calculation and skill gap analysis

## Key Features

- **Configurable Company List:** Companies managed via CSV file
- **Priority-Based Analysis:** Different analysis depth based on company priority
- **Multi-Criteria Matching:** Skills, experience, education, location
- **Skill Gap Identification:** Identifies missing skills with recommendations
- **Error Handling:** Graceful degradation and user-friendly error messages
