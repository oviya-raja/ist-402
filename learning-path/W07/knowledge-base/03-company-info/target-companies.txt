PRIMARY TARGET COMPANIES
========================

1. CISCO
--------
Company Overview:
- Industry: Networking, Cybersecurity, Cloud
- Size: Large (80,000+ employees)
- Headquarters: San Jose, California
- Culture: Innovation-focused, collaborative

Job Posting URL: https://careers.cisco.com/global/en

Typical Roles Available:
- Software Engineer
- Network Software Engineer
- Cloud Solutions Engineer
- Security Software Engineer
- DevOps Engineer

Common Requirements:
- Programming: Python, Java, C/C++, Go
- Networking: TCP/IP, routing protocols
- Cloud: AWS, Azure, GCP
- Tools: Docker, Kubernetes
- Education: BS in CS/EE or related

Typical Entry-Level Requirements:
- BS in Computer Science or related
- 0-2 years experience
- Strong programming fundamentals
- Interest in networking/cloud

2. SAP
------
Company Overview:
- Industry: Enterprise Software, ERP, Cloud
- Size: Large (100,000+ employees)
- Headquarters: Walldorf, Germany (US offices in multiple cities)
- Culture: Enterprise-focused, global

Job Posting URL: https://jobs.sap.com/

Typical Roles Available:
- Software Development Engineer
- Cloud Platform Engineer
- Full-Stack Developer
- Data Engineer

Common Requirements:
- Programming: Java, JavaScript, Python, ABAP
- Databases: SAP HANA, SQL
- Cloud: SAP Cloud Platform, AWS
- Education: BS in CS or related

Typical Entry-Level Requirements:
- BS in Computer Science
- 0-2 years experience
- Java or JavaScript proficiency
- Willingness to learn SAP technologies

3. GOOGLE
---------
Company Overview:
- Industry: Technology, Search, Cloud, AI
- Size: Very Large (180,000+ employees)
- Headquarters: Mountain View, California
- Culture: Innovation, technical excellence, data-driven

Job Posting URL: https://careers.google.com/jobs/

Typical Roles Available:
- Software Engineer
- Associate Software Engineer
- Software Engineer I/II
- Site Reliability Engineer (SRE)

Common Requirements:
- Programming: C++, Java, Python, Go
- Algorithms: Strong data structures/algorithms
- System Design: Distributed systems
- Education: BS/MS/PhD in CS or related

Typical Entry-Level Requirements:
- BS in Computer Science (MS/PhD preferred for some roles)
- Strong algorithms and data structures
- System design knowledge
- Coding interview performance critical

Interview Process:
- Technical phone screen
- On-site interviews (algorithm, system design, coding)
- Emphasis on problem-solving and technical depth
