## How to Capture Screenshots

1. Open the workflow URL in your browser
2. Sign in to OpenAI platform
3. Navigate to each section
4. Capture screenshot (Cmd+Shift+4 on Mac, Snipping Tool on Windows)
5. Save with the exact filename listed above
6. Store in this `screenshots/` directory

## Notes

- All screenshots should show the actual Agent Builder interface
- Ensure workflow URL is visible in browser address bar (if possible)
- Screenshots should be clear and readable
- Include full context of each configuration panel
